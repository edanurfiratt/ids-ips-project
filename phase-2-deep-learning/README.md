# Phase 2 – Deep Learning Based IDS/IPS Development

This phase represents the second stage of the project, focusing on improving the IDS/IPS system using custom-generated data and deep learning techniques.

---

## 🎯 Objective

To develop a more advanced anomaly detection system by using only original network traffic data and integrating deep learning models.

---

## 🧠 Key Improvements

- Use of custom-generated network traffic instead of ready datasets  
- Expansion of dataset with various attack scenarios  
- Integration of deep learning models (ANN, CNN, LSTM)  
- Development of a hybrid system combining ML and DL approaches  

---

## ⚙️ Development Plan

### 🔍 Stage 1 – Research & Setup
- Study deep learning approaches  
- Install TensorFlow / PyTorch  
- Prepare system infrastructure  

### 📊 Stage 2 – Data Expansion
- Generate new traffic data  
- Increase diversity of normal and attack traffic  
- Improve dataset quality  

### 🧠 Stage 3 – Model Training
- Train machine learning and deep learning models  
- Apply hyperparameter optimization  
- Select best-performing model  

### 🛡️ Stage 4 – IPS Integration
- Develop automatic attack detection and prevention system  
- Integrate AI-based IPS module  
- Perform system testing  

### 📄 Stage 5 – Finalization
- Prepare documentation and final report  
- Complete project presentation  

---

## 📊 Expected Outcomes

- Improved anomaly detection performance  
- Better generalization using real-world data  
- Real-time detection and prevention capability
